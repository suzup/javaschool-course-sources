# Spring Boot 전에 끝내는 HTML·JavaScript

JavaSchool HTML·JavaScript 과정 8강을 마쳤을 때의 최종 소스입니다.

## 실행 환경

- Chrome 또는 Edge
- 인터넷 연결

## 실행

1. 이 폴더의 `index.html`을 Chrome 또는 Edge로 엽니다.
2. JavaSchool 공개 API에서 첫 문제가 나타나는지 확인합니다.
3. 1부터 4까지의 정답 번호를 입력하고 `정답 보내기`를 누릅니다.

이 소스는 다음 학습용 공개 API를 사용합니다.

```text
https://api.javaschool.org/api/learning/v1/quiz
```

현재 점수는 브라우저에만 저장되므로 페이지를 새로 고치면 0점으로 돌아갑니다. 다음 Spring Boot 과정에서는 이 API 주소를 직접 만든 서버 주소로 바꿉니다.
