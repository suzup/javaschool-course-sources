# 직접 만들고 플레이하는 Java 객체지향 퀴즈

JavaSchool Java 객체지향 퀴즈 과정 9강을 마쳤을 때의 최종 소스입니다.

## 실행 환경

- Java 17 이상
- VS Code와 Extension Pack for Java

## 실행

VS Code에서 이 폴더를 열고 터미널에서 실행합니다.

```powershell
javac *.java
java Main
```

닉네임을 입력해 객관식과 OX 문제를 풀 수 있습니다. 모든 답을 제출한 뒤 문제별 정답 여부, 내 답, 해설과 100점 기준 점수를 보여 줍니다.

컴파일 뒤 만들어지는 `.class` 파일은 강의 소스에 포함하지 않습니다.
