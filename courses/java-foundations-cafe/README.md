# 카페 주문 프로그램으로 배우는 Java 기초

JavaSchool Java 기초 과정 16강을 마쳤을 때의 최종 소스입니다.

## 실행 환경

- Java 17 이상
- VS Code와 Extension Pack for Java

## 실행

VS Code에서 이 폴더를 열고 터미널에서 실행합니다.

```powershell
javac *.java
java Main
```

메뉴 번호와 수량을 입력하면 장바구니에 메뉴가 담기고 총금액이 표시됩니다. 같은 메뉴를 다시 담기, 잘못된 입력 다시 받기와 주문 확정까지 포함합니다.

컴파일 뒤 만들어지는 `.class` 파일은 강의 소스에 포함하지 않습니다.
