# JavaSchool 강의 소스

JavaSchool 학습 과정에서 사용하는 실행 소스를 모아 둔 저장소입니다.

강의 원고는 JavaSchool 사이트에서 읽고, 이 저장소에서는 시작 소스와 완성 소스를 확인합니다. 과정마다 `courses` 아래에 독립된 폴더를 사용합니다.

## 과정 목록

| 과정 | 소스 폴더 | 실행 환경 |
|---|---|---|
| Java 퀴즈 서버를 직접 만드는 Spring Boot 입문 | [`courses/spring-boot-quiz`](courses/spring-boot-quiz) | Java 17, Spring Boot 3.5.16, MySQL 8.0 |

## 소스 사용 방법

각 과정 폴더의 `README.md`에서 준비물과 실행 명령을 확인하세요. 강의별 시작·완성 소스는 과정 제작이 끝난 뒤 태그와 내려받기 파일로 제공합니다.

태그 이름은 다음 규칙을 사용합니다.

```text
과정이름-v1-lesson-강의번호
```

예시:

```text
spring-boot-quiz-v1-lesson-06
```

실제 DB 비밀번호, 개인 환경 설정과 빌드 결과는 저장소에 올리지 않습니다.
